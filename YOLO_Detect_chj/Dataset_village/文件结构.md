# Dataset_village 文件结构

## 数据集概述
这是一个用于YOLO目标检测的乡村文档数据集，包含训练集和验证集两部分。

## 数据集统计信息
- 训练集图片数量: 100+ 张
- 验证集图片数量: 100+ 张
- 训练集标注数量: 100+ 个
- 验证集标注数量: 100+ 个

## 类别定义
数据集共包含 7 个类别：
| 序号 | 类别名 | 说明 |
|------|--------|------|
| 0 | title | 标题 |
| 1 | caption | 图注 |
| 2 | txt_1 | 文本1 |
| 3 | txt_2 | 文本2 |
| 4 | img | 图片 |
| 5 | txt_3 | 文本3 |
| 6 | txt_4 | 文本4 |

## 完整目录树
```
Dataset_village/
├── images/
│   ├── train/
│   │   ├── Page_022.png
│   │   ├── Page_023.png
│   │   ├── ...
│   └── val/
│       ├── Page_032.png
│       ├── Page_033.png
│       └── ...
├── labels/
│   ├── train/
│   │   ├── (对应训练集图片的YOLO标注文件)
│   │   └── ...
│   └── val/
│       ├── Page_032.txt
│       ├── Page_033.txt
│       └── ...
├── village_dataset.yaml
└── classes.txt
```

## 文件说明
1. **village_dataset.yaml**: YOLO数据集配置文件，定义数据集路径和类别信息
2. **classes.txt**: 类别名称列表文件
3. **images/**: 存放所有图片的根目录
   - train/: 训练集图片（PNG格式）
   - val/: 验证集图片（PNG格式）
4. **labels/**: 存放所有标注文件的根目录
   - train/: 训练集标注（YOLO格式TXT文件）
   - val/: 验证集标注（YOLO格式TXT文件）

## 标注格式说明
标注文件采用YOLO格式，每一行表示一个目标，格式为：
```
<class_id> <x_center> <y_center> <width> <height>
```
其中坐标值均为归一化后的相对坐标（范围0-1）。
