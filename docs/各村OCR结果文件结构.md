# 各村OCR结果文件结构
## 1. 各村OCR结果目录层级规则
```
各村OCR结果/
├── [序号]_[村名]/
│   ├── [序号]_[村名].txt   # 该村正文描述文档
│   └── （图片文件1..N）     # 该村的插图
├── [序号]_[村名]/
│   ├── [序号]_[村名].txt
│   └── （图片文件1..N）
└── ...
```

各村OCR结果/
├── 1_大围村/

│   └── 陈济棠别墅（摄于2016年，温泉镇提供）.png
